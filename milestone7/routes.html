FAQ

How do I create a route?

Go to the Create page to create a route. Make sure you have a good name in mind, a rough estimate of the grade, and a short description for the route. The more routes on Route Mapper the better and the more places that people can go to have some fun climbing.

How do I save a route?

Either on the Create page or Explore page when you click on a marker there is a button to save. When you save the button should turn blue letting you know that it saved correctly.

How can I view routes on the map?

The Explore page is the main place to view routes on the map. You can go pretty much anywhere and look for markers, and simply by clicking on a marker you can see some information about that route.

How do I edit a route I already created?

To edit a route that you created simply go back into the Create page and click on the marker for the route you want to edit. After that just put in the information you want to change and push create again.

Where can I give feedback on the site?

go to the Contact page and send us an email, give us a call, or fill out the form. Please use this to report any bugs you find or if you simply want to give some feedback on the site. Feedback is always important to Route Mapper as we are constantly striving to improve the site for a better user experience.

What is Route Mapper?

In short Route Mapper is a website that was built to make getting into climbing easier and more fun for everyone. Go to the About page to learn about Rotue Mapper.
